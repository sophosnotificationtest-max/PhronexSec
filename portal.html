1
Criar conta no Render
Acesse render.com e crie uma conta gratuita. Pode usar login com GitHub.
2
Criar banco PostgreSQL
No dashboard do Render, clique em New → PostgreSQL. Selecione o plano gratuito e anote a Internal Database URL.
3
Deploy do Peppermint via Docker
Clique em New → Web Service → Deploy an existing image e use:
pepperlabs/peppermint:latest
4
Variáveis de ambiente
Em Environment, adicione:
DATABASE_URL = (cole a URL do PostgreSQL) SECRET = qualquer-string-longa-aleatoria PORT = 3000
5
Configurar porta e fazer deploy
Em Settings defina a porta 3000. Clique em Deploy e aguarde ~3 min. O Render gera uma URL pública automática.
6
Colar a URL nesta página
Copie a URL gerada (ex: https://phronexsec-suporte.onrender.com), cole no campo acima e clique Salvar. O botão será ativado automaticamente.
7
Criar usuários para os clientes
No Peppermint, acesse Admin → Users → Create e crie um login para cada cliente. Login padrão inicial: admin@admin.com / 1234
ℹ Plano gratuito do Render tem cold start (~30s na 1ª abertura após inatividade). Para manter ativo, use UptimeRobot (gratuito) pingando a URL a cada 5 min, ou considere o plano Starter ($7/mês).
